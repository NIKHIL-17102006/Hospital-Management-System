# Advanced SQL: Indexes and Views

This module enhances the performance and usability of the database.

## Indexes
- Used to improve query performance
- Reduce search time on large tables
- Applied on frequently queried columns

## Views
- Virtual tables based on SQL queries
- Simplify complex queries
- Improve security by restricting direct table access

## Benefits
- Faster data retrieval
- Cleaner query structure
- Better database management

This section demonstrates practical optimization techniques used in real-world databases.
